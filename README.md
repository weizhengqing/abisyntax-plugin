# ABINIT VS Code Extension

This extension provides syntax highlighting and language support for ABINIT input (`.abi`, `.in`) and output (`.abo`, `.output`, `DEN`, `GSR`) files.

## Features

- Syntax highlighting for ABINIT keywords, variables, numbers, comments, and strings.
- Auto-closing pairs and bracket matching.

## Installation

1. Download the generated `.vsix` file.
2. Open VS Code.
3. Go to the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
4. Click on the `...` menu and select "Install from VSIX...".
5. Select the downloaded `.vsix` file.
